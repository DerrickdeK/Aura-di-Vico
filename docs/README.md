# Guide Aura (Italiano) — Indice

Tre guide brevi, una per ogni ruolo. Condividile digitalmente con chi
ne ha bisogno, a seconda dello status dell'utente.

| File | A chi è destinata | Quando condividerla |
|---|---|---|
| **[GUIDA_VISITATORE.md](GUIDA_VISITATORE.md)** | Visitatori e nuovi utenti curiosi | Quando qualcuno apre Aura per la prima volta. È la guida più leggera (1–2 pagine). |
| **[GUIDA_CONTRIBUTORE.md](GUIDA_CONTRIBUTORE.md)** | Studenti, curatori, cittadini che vogliono arricchire la città | Quando crei un account come contributore. Spiega tipologie di contributo, regole di scrittura, processo di moderazione (2–3 pagine). |
| **[GUIDA_ADMIN.md](GUIDA_ADMIN.md)** | Admin e co-admin che gestiscono la città | Solo a chi ha o sta per ricevere accesso admin. Copre POI, area config, moderazione, clonazione su nuove città (4–5 pagine). |

Le guide sono in Markdown — si leggono già bene così su GitHub, ma
puoi anche convertirle in PDF (es. con `pandoc`) o stamparle.
